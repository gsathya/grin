Setting up Grin

Grin is an addon that enables you to support the Mozilla Foundation at no aditional cost to you using the Amazon Smile program.

Without further set up, this addon will ensure that every time you load an *.amazon.com domain name you will be redirected to the smile program, so that any purchase you make will result in a small donation. In order to set the Mozilla Foundation as your charity of choice, you'll need to log in to Amazon and select the Mozilla Foundation as your charity.
